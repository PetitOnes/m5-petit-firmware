🤖 M5 Petit — ブラウザから書き込み

M5Stack CoreS3にぷちのファームウェアを書き込みます。Arduino IDEのインストールは不要です。
Flash the M5 Petit firmware to your M5Stack CoreS3 — no Arduino IDE needed.

用意するもの / You need
  1. M5Stack CoreS3 と USB-Cケーブル（データ通信対応のもの） / and a USB-C data cable
  2. Chrome または Edge ブラウザ（Safari・Firefoxは非対応） / Chrome or Edge (Safari & Firefox are not supported)
1. 書き込み / Flash

CoreS3をUSBでPCにつなぎ、ボタンを押してポートを選ぶ / Connect the CoreS3 via USB, click the button and pick the serial port

初めての書き込みでは「Erase device」にチェックを入れてください / Check "Erase device" on first install

2. スマホから設定 / Set up from your phone

初回起動時にCoreS3の画面にQRコードが表示されます。スマホで読み取って表示されたWiFi（M5Petit-XXXX）に接続すると設定ページが開くので、WiFi・名前・色を入力して保存してください（SDカードへの書き込みは不要です）。
A QR code appears on the CoreS3 screen. Scan it, join the M5Petit-XXXX WiFi it shows, and fill in WiFi/name/color on the page that opens (no SD card editing needed).

これはM5自身が一時的に作るWiFi（アクセスポイント）に接続して行うので、このページの外、スマホの中で完結します。
This step happens on a WiFi network the M5 itself creates temporarily, so it's done on your phone, outside this page.

3. SDアセット送信（任意） / Push SD assets (optional)

顔画像・効果音をSDカードに入れたい場合、ここから直接送れます。空のFAT32 SDカードをM5に挿しておいてください。
If you want custom face images / sounds on the SD card, push them right here. Insert an empty FAT32 SD card into the M5 first.

送信先IP: 手順2完了直後は画面右下（30秒間）に緑色で表示されます。またはキャラクターID（ホスト名）で <キャラクターID>.local でも指定できます。
Target: right after step 2, the IP shows in green in the bottom-right corner of the screen for 30s. Or use the hostname, <character-id>.local.
