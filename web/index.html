🤖 M5 Petit — ブラウザから書き込み

M5Stack CoreS3にぷちのファームウェアを書き込みます。Arduino IDEのインストールは不要です。
Flash the M5 Petit firmware to your M5Stack CoreS3 — no Arduino IDE needed.

用意するもの / You need
  1. M5Stack CoreS3 と USB-Cケーブル（データ通信対応のもの） / and a USB-C data cable
  2. Chrome または Edge ブラウザ（Safari・Firefoxは非対応） / Chrome or Edge (Safari & Firefox are not supported)

1. CoreS3をUSBでPCにつなぐ / Connect the CoreS3 via USB

2. ボタンを押してポートを選ぶ / Click and pick the serial port

初めての書き込みでは「Erase device」にチェックを入れてください / Check "Erase device" on first install

書き込みが終わったら / After flashing

初回起動時にCoreS3の画面にQRコードが表示されます。スマホで読み取って表示されたWiFiに接続し、 開いた設定ページでWiFi・名前・色を入力してください（SDカードへの書き込みは不要です）。
On first boot, a QR code appears on the CoreS3 screen. Scan it with your phone, join the WiFi it shows, and fill in WiFi/name/color on the page that opens (no SD card editing needed).

顔画像・効果音を使いたい場合はSDカードに入れてください。手順はセットアップガイドへ：
If you want custom face images / sounds, put them on the SD card. Full guide:

→ M5Stack CoreS3 セットアップ手順 / Setup Guide

うまくいかないとき: ケーブルが充電専用でないか確認 / ポートが出ないときはドライバ不要のはずですがOSを再起動してみてください。
